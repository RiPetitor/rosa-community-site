Установка

Пошаговое руководство по установке ROSA Linux

Эта страница поможет вам установить ROSA Linux на компьютер или ноутбук.

Системные требования

КомпонентМинимумРекомендуется
Процессор64-bit (x86_64)Многоядерный
ОЗУ2 ГБ4 ГБ и более
Диск20 ГБ50 ГБ и более
ВидеоСовместимое с LinuxС поддержкой 3D

Скачивание образа

  1. Перейдите на страницу загрузки.
  2. Выберите редакцию (ROSA Fresh, LTS или Server).
  3. Скачайте ISO-образ.
  4. Проверьте контрольную сумму, чтобы убедиться в целостности файла.

Создание загрузочного носителя

Linux

Используйте dd:

sudo dd if=ROSA.FRESH.iso of=/dev/sdX bs=4M status=progress
sync

Замените /dev/sdX на устройство вашей флешки (например, /dev/sdb).

Windows

Рекомендуемые программы:

  • Rufus — простой и надёжный.
  • balenaEtcher — кроссплатформенный.

macOS

diskutil list  # найдите вашу флешку
sudo diskutil unmountDisk /dev/diskN
sudo dd if=ROSA.FRESH.iso of=/dev/rdiskN bs=4m

Установка

После загрузки с USB вы увидите рабочий стол ROSA Linux в режиме Live. Это позволяет протестировать систему без установки и запустить установщик.

Запуск установщика

Дважды кликните «Установить ROSA» на рабочем столе.

Основные шаги

  1. Язык и раскладка — выберите русский интерфейс и нужную раскладку.
  2. Часовой пояс — укажите свой регион и город.
  3. Разметка диска — автоматически или вручную.
  4. Пользователь — создайте учётную запись и пароль.
  5. Подтверждение — проверьте параметры и начните установку.
Внимание
Перед изменением разделов сделайте резервную копию важных данных.

После установки

Перезагрузите компьютер, извлеките USB-флешку и перейдите к первоначальной настройке.

Возможные проблемы

Компьютер не загружается с USB

  • Проверьте настройки UEFI/BIOS.
  • Отключите Secure Boot.
  • Попробуйте другой USB-порт.

Установщик не видит диск

  • Проверьте подключение накопителя.
  • Убедитесь, что диск не зашифрован другой ОС.
  • Включите режим AHCI вместо RAID.
