Управление пакетами

Как устанавливать, обновлять и удалять программы

ROSA Linux использует пакетный менеджер DNF для управления программным обеспечением.

Основные команды

# поиск пакета
dnf search имя_пакета

# установка пакета
sudo dnf install имя_пакета

# удаление пакета
sudo dnf remove имя_пакета

# обновление системы
sudo dnf update

Работа с репозиториями

# показать подключенные репозитории
dnf repolist

# добавить репозиторий (пример)
sudo dnf config-manager --add-repo https://example.repo

Полезные операции

# история операций
dnf history

# очистка кеша
sudo dnf clean all
Графические инструменты
В KDE можно использовать Discover, в GNOME — GNOME Software.

Flatpak

# включить Flathub
sudo flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

Советы

Внимание
Не смешивайте пакеты из разных версий ROSA Linux — это может привести к конфликтам.
