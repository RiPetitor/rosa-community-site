★ СКАЧАТЬ РОСА GNU/LINUX ★

Выберите подходящую редакцию операционной системы. Все образы доступны бесплатно и без регистрации.

ROSA Fresh

Версия 13.1
Описание

Для дома и повседневной работы. Актуальные версии ядра, окружений рабочего стола и ключевых приложений.

Для чего
  • Рабочий стол KDE Plasma 6 или GNOME
  • Офисный пакет LibreOffice
  • Браузер Firefox
  • Мультимедиа: VLC, видеоредакторы
  • Графический установщик драйверов
  • Работа без командной строки
Требования: 4 ГБ ОЗУ, 25 ГБ на диске, 64-bit процессор

ROSA Fresh LTS

Версия 12.5.1
Описание

Для тех, кому важна стабильность. Рекомендуется для тех, кому важна стабильность.

Для чего
  • KDE Plasma 5, GNOME, LXQt, XFCE
  • Долгосрочная поддержка
  • Проверенные версии ПО
  • Поддержка 32-bit (LXQt)
  • Офисный пакет LibreOffice
  • Полный набор приложений
Требования: 2 ГБ ОЗУ, 20 ГБ на диске

ROSA Fresh Server

Версия 13.1
Описание

Для серверных задач и лабораторий. Экспериментальная ветка для серверных задач.

Для чего
  • Минимальный образ без GUI
  • Веб-сервер Apache/Nginx
  • Базы данных PostgreSQL, MariaDB
  • Контейнеризация Docker, Podman
  • Виртуализация KVM/QEMU
  • Средства администрирования
Требования: 1 ГБ ОЗУ (мин.), 10 ГБ на диске

Community

BFRESH (Barium)

BFRESH — комьюнити-сборка, объединяющая идеи Barium и ROSA Fresh. Текущие образы выпускаются для платформы ROSA 13 и распространяются в форматах ISO и tar.gz. Проект ещё не в бета, но уже доступен для тестирования.

  • Гибрид Barium + Fresh для экспериментов
  • Свободные сборки вместо прежних «free» Barium
  • ISO и tar.gz для тестирования
  • Проект развивается сообществом

Подробные инструкции и сценарии использования смотрите в wiki.

★ Как установить ★

Шаг 1

Скачайте образ

Выберите подходящую редакцию и скачайте ISO-образ. Рекомендуем проверить контрольную сумму файла.

Шаг 2

Создайте носитель

Запишите образ на USB-флешку с помощью Etcher, Rufus или dd.

Шаг 3

Установите систему

Загрузитесь с USB и следуйте инструкциям установщика. Подробности в руководстве.

★ Частые вопросы ★

Какую версию выбрать?

РОСА Фреш 13.1 — для энтузиастов, которые хотят новейшее ПО. РОСА Фреш LTS 12.5.1 — для тех, кому важна стабильность.

Можно ли попробовать без установки?

Да! Все образы поддерживают режим Live — загрузитесь с флешки и попробуйте систему без изменения данных на диске.

Где получить помощь?

Посетите форум РОСА, Telegram-канал или Wiki.

Как проверить контрольную сумму?

На странице загрузки рядом с каждым образом указана контрольная сумма. Сверьте её командой sha256sum имя_файла.iso или md5sum имя_файла.iso.
