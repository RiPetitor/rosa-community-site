ROSA Linux — это российский дистрибутив GNU/Linux, разработанный компанией НТЦ ИТ РОСА. Система создаётся для широкого круга пользователей: от домашних компьютеров до корпоративных серверов.

## История

Проект ROSA Linux основан в 2009 году. За это время дистрибутив прошёл путь от форка Mandriva Linux до полностью самостоятельной системы с собственной инфраструктурой сборки и репозиториями.

## Последние версии

- **ROSA «ФРЕШ» 13**: Домашний дистрибутив на основе новой платформы ROSA 13.
- **Релиз 13.1** для домашних пользователей.
- **Ядро 6.12 LTS** обеспечивает поддержку новой аппаратуры.
- **glibc 2.40** для совместимости с переносимыми приложениями.
- **systemd 256** — современная система управления службами.

## Основные особенности

- **Русская локализация** — интерфейс и документация на русском языке.
- **Собственные репозитории** — пакеты собираются и тестируются командой ROSA.
- **Поддержка российского ПО** — совместимость с отечественными программами и сертификатами.
- **Несколько редакций** — от легковесного десктопа до серверных решений.
- **Упрощенная нумерация**: Мажорный номер версии фреша совпадает с номером платформы.
- **Новейшие версии библиотек и пользовательских программ**.
- **Обновленный дизайн** в традиционной цветовой гамме.
- **Базовая поддержка китайского и арабского языков**.

## Технические основы

ROSA Linux базируется на пакетном менеджере **DNF** (ранее использовался urpmi). Система использует формат пакетов **RPM**.

| Компонент | Технология |
| --- | --- |
| Пакетный менеджер | DNF |
| Формат пакетов | RPM |
| Система инициализации | systemd |
| Ядро | Linux 6.12 LTS |
| glibc | 2.40 |
| systemd | 256 |

## Особенности установки

- **Быстрая установка без лишних вопросов**.
- **Набор готовых решений** с различными графическими окружениями.
- **Поддержка нестандартных Wi-Fi адаптеров «из коробки»**.
- **Автоустановка драйверов для видеокарт NVIDIA**.

## Сообщество

Помимо коммерческой разработки, вокруг ROSA Linux сформировалось сообщество пользователей и энтузиастов. Сообщество помогает с документацией, тестированием и поддержкой.
