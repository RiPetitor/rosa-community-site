## Как правильно задавать вопросы

Чтобы получить помощь быстрее, предоставьте:

1. **Версию системы**
   ```bash
   cat /etc/os-release
   ```

2. **Описание проблемы** — что делали, что ожидали, что получили.

3. **Логи и вывод команд** — если есть ошибки.
   ```bash
   journalctl -xe | tail -50
   ```

4. **Информацию об оборудовании** (если проблема с драйверами).
   ```bash
   inxi -Fxz
   ```

<div class="warning">
  <div class="title">Важно</div>
  Не публикуйте личные данные и пароли в логах.
</div>

## Самопомощь

Перед обращением попробуйте:

- **Поиск в интернете** — опишите ошибку в поисковике.
- **Чтение логов** — загляните в `/var/log/` или используйте `journalctl`.
- **Раздел «Решение проблем»** — в этой документации собраны типичные ситуации.
