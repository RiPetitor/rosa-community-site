ROSA Linux выпускается в нескольких редакциях, каждая из которых ориентирована на определённые задачи.

## ROSA Fresh

Редакция для домашних пользователей и энтузиастов.

**Особенности:**
- Актуальные версии программ
- Поддержка современного оборудования
- Рабочие окружения: KDE Plasma, GNOME, LXQt

**Рекомендуется для:**
- Домашних компьютеров
- Изучения Linux
- Разработчиков, которым важны свежие пакеты

## ROSA Fresh LTS

Версия с длительной поддержкой (Long Term Support).

**Особенности:**
- Стабильные и проверенные пакеты
- Обновления безопасности в течение нескольких лет
- Меньше изменений между обновлениями

**Рекомендуется для:**
- Рабочих станций, где важна стабильность
- Пользователей, которые не хотят частых обновлений

## ROSA Server

Серверная редакция без графического интерфейса.

**Особенности:**
- Минимальная установка
- Оптимизация для серверных задач
- Инструменты администрирования

**Рекомендуется для:**
- Веб-серверов
- Файловых хранилищ
- Виртуализации

## Сравнение редакций

| Характеристика | Fresh | Fresh LTS | Server |
| --- | --- | --- | --- |
| Обновления пакетов | Частые | Редкие | Редкие |
| Срок поддержки | ~1 год | 3-5 лет | 3-5 лет |
| Графический интерфейс | Да | Да | Нет |
| Целевая аудитория | Энтузиасты | Стабильность | Серверы |

## Какую редакцию выбрать

<div class="tip">
  <div class="title">Для новичков</div>
  Начните с <strong>ROSA Fresh</strong> — это наиболее универсальный вариант с актуальным ПО и хорошей поддержкой оборудования.
</div>

<div class="info">
  <div class="title">Для работы</div>
  Если стабильность важнее новизны — выберите <strong>ROSA Fresh LTS</strong>.
</div>
