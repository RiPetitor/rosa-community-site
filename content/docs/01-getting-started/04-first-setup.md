После установки стоит сделать несколько важных шагов — это улучшит безопасность
и удобство работы.

## Обновление системы

Откройте «Центр обновлений» или выполните в терминале:

```bash
sudo dnf update
```

## Драйверы и кодеки

### Видеокарты NVIDIA

ROSA Linux автоматически устанавливает драйверы для поддерживаемых видеокарт NVIDIA. Если автоматическая установка не произошла, откройте «Центр приложений» → «Драйверы» и установите рекомендованный пакет.

### Видеокарты AMD

Драйверы AMDGPU уже включены в ядро 6.12 LTS. Для OpenGL/Vulkan обновляйте систему
регулярно.

### Мультимедиа

Установите кодеки и дополнительные форматы:

```bash
sudo dnf install ffmpeg gstreamer1-plugins-bad-free gstreamer1-plugins-ugly
```

## Язык и раскладки

### KDE Plasma

1. Откройте «Параметры системы».
2. Перейдите в **Устройства ввода → Клавиатура**.
3. Добавьте раскладки и настройте переключение (например, `Alt+Shift`).

### GNOME

1. Откройте «Настройки».
2. Перейдите в раздел **Клавиатура**.
3. Добавьте источники ввода.

## Установка полезных программ

- Офис: **LibreOffice** (обычно уже установлен).
- Браузер: **Firefox**, **Chromium** или **Яндекс.Браузер**.
- Мультимедиа: **VLC**.

```bash
sudo dnf install libreoffice vlc
```

## Резервное копирование

Для снимков системы можно использовать **Timeshift**:

```bash
sudo dnf install timeshift
```

<div class="tip">
  <div class="title">Совет</div>
  Настройте резервные копии перед установкой крупных обновлений.
</div>

## Особенности ROSA Linux

- **Поддержка современного оборудования**: Благодаря ядру 6.12 LTS и glibc 2.40, система поддерживает новое оборудование.
- **Обновленный дизайн**: В ROSA Fresh 13 реализован обновленный дизайн в традиционной цветовой гамме.
- **Базовая поддержка китайского и арабского языков**: В системе реализована базовая поддержка дополнительных языков.
