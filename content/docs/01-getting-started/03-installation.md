Эта страница поможет вам установить ROSA Linux на компьютер или ноутбук.

## Системные требования

| Компонент | Минимум | Рекомендуется |
| --- | --- | --- |
| Процессор | 64-bit (x86_64) | Многоядерный |
| ОЗУ | 2 ГБ | 4 ГБ и более |
| Диск | 20 ГБ | 50 ГБ и более |
| Видео | Совместимое с Linux | С поддержкой 3D |
| Поддержка EFI/UEFI | Да | Да |

## Скачивание образа

1. Перейдите на [страницу загрузки](@/download/_index.md).
2. Выберите редакцию (ROSA Fresh 13, LTS или Server).
3. Скачайте ISO-образ.
4. Проверьте контрольную сумму, чтобы убедиться в целостности файла.

## Создание загрузочного носителя

### Linux

Используйте `dd`:

```bash
sudo dd if=ROSA.FRESH.iso of=/dev/sdX bs=4M status=progress
```

Замените `/dev/sdX` на устройство вашей флешки (например, `/dev/sdb`).

### Windows

Рекомендуемые программы:

- [Rufus](https://rufus.ie/) — простой и надёжный.
- [balenaEtcher](https://etcher.balena.io/) — кроссплатформенный.

### macOS

```bash
diskutil list  # найдите вашу флешку
sudo diskutil unmountDisk /dev/diskN
sudo dd if=ROSA.FRESH.iso of=/dev/rdiskN bs=4m
```

## Установка

После загрузки с USB вы увидите рабочий стол ROSA Linux в режиме Live. Это
позволяет протестировать систему без установки и запустить установщик.

### Запуск установщика

Дважды кликните **«Установить ROSA»** на рабочем столе.

### Основные шаги

1. **Язык и раскладка** — выберите русский интерфейс и нужную раскладку.
2. **Часовой пояс** — укажите свой регион и город.
3. **Разметка диска** — автоматически или вручную.
4. **Пользователь** — создайте учётную запись и пароль.
5. **Лицензия** — Примите условия лицензии
6. **Подтверждение** — проверьте параметры и начните установку.

## Особенности установки в ROSA Linux

- **Быстрая установка без лишних вопросов** — процесс установки оптимизирован для удобства.
- **Набор готовых решений** — выбор различных графических окружений при установке.
- **Поддержка нестандартных Wi-Fi адаптеров «из коробки»** — драйвера уже включены в образ.
- **Автоустановка драйверов для видеокарт NVIDIA** — при наличии поддерживаемой видеокарты драйвера устанавливаются автоматически.

## После установки

Перезагрузите компьютер, извлеките USB-флешку и перейдите к
[первоначальной настройке](@/docs/01-getting-started/04-first-setup.md).

## Возможные проблемы

### Компьютер не загружается с USB

- Проверьте настройки UEFI/BIOS.
- Отключите Secure Boot.
- Попробуйте другой USB-порт.

### Установщик не видит диск

- Проверьте подключение накопителя.
- Убедитесь, что диск не зашифрован другой ОС.
- Включите режим AHCI вместо RAID.

### Проблемы с загрузкой после установки

- Проверьте, что UEFI/BIOS настроен правильно (режим UEFI или Legacy).
- Убедитесь, что GRUB установлен в правильное место (обычно это MBR или EFI раздел).
