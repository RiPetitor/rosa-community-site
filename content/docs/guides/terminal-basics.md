Терминал — мощный инструмент для работы с Linux. Начнём с базовых команд.

## Где открыть терминал

- **KDE Plasma**: `Ctrl+Alt+T` или приложение Konsole.
- **GNOME**: `Ctrl+Alt+T` или приложение Terminal.

## Навигация по системе

```bash
pwd        # где я нахожусь
ls -la     # содержимое каталога
cd /path   # переход в каталог
```

## Работа с файлами

```bash
mkdir проекты
cp файл.txt копия.txt
mv старое.txt новое.txt
rm файл.txt
```

<div class="danger">
  <div class="title">Осторожно</div>
  Команда <code>rm</code> удаляет файлы безвозвратно. Не используйте <code>rm -rf /</code>.
</div>

## Поиск

```bash
# поиск файлов
find ~/Документы -name "*.pdf"

# поиск текста внутри файлов
grep -R "строка" ~/Документы
```

## Права доступа

```bash
ls -l
chmod +x скрипт.sh
```

## Горячие клавиши

| Комбинация | Действие |
| --- | --- |
| Ctrl+C | Прервать выполнение |
| Ctrl+L | Очистить экран |
| Ctrl+A | В начало строки |
| Ctrl+E | В конец строки |
| Tab | Автодополнение |
