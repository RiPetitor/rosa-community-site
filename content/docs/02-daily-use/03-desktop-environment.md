ROSA Linux поддерживает несколько рабочих окружений. Выбор зависит от ваших предпочтений и возможностей компьютера.

## KDE Plasma

Мощное и настраиваемое окружение. Используется по умолчанию в ROSA Fresh 13.

### Особенности

- Гибкая настройка внешнего вида
- Панели, виджеты, эффекты
- Интеграция с приложениями KDE (Dolphin, Kate, Konsole)
- Поддержка современных технологий (Wayland, OpenGL 4.x)

### Настройка

1. Откройте «Параметры системы».
2. Разделы: Внешний вид, Рабочий стол, Окна.

### Горячие клавиши

| Комбинация | Действие |
| --- | --- |
| `Meta` (Win) | Открыть меню |
| `Meta + E` | Файловый менеджер |
| `Meta + L` | Заблокировать экран |
| `Alt + Tab` | Переключение окон |
| `Ctrl + Alt + T` | Терминал (если настроено) |

## GNOME

Минималистичное окружение с фокусом на простоту.

### Особенности

- Чистый интерфейс без отвлекающих элементов
- Поиск через Activities
- Расширения для дополнительных функций
- Поддержка Wayland сеансов

### Настройка

1. Откройте «Настройки».
2. Для расширений используйте GNOME Extensions или Tweaks.

```bash
sudo dnf install gnome-tweaks gnome-extensions-app
```

## LXQt

Лёгкое окружение для слабых компьютеров.

### Особенности

- Низкое потребление памяти
- Быстрый запуск
- Классический интерфейс
- Совместимость с новыми версиями Qt

## Смена окружения

### При входе в систему

На экране входа (SDDM, GDM) выберите сессию перед вводом пароля.

### Установка другого окружения

```bash
# KDE Plasma
sudo dnf in sddm

# GNOME
sudo dnf in gdm

# LXQt
sudo dnf in lightdm
```

  <div class="title">Совет</div>

## Темы и оформление

### KDE Plasma

- Параметры системы → Внешний вид → Глобальная тема
- Можно скачать темы из интернета прямо в настройках
- ROSA Fresh 13 включает обновленный дизайн в традиционной цветовой гамме

### GNOME

- Используйте GNOME Tweaks для смены тем
- Темы GTK размещаются в `~/.themes/`
