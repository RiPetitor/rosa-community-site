Для видеокарт NVIDIA рекомендуется использовать проприетарные драйверы для получения максимальной производительности в играх и графических приложениях. В ROSA Linux для этого есть утилита `kroko-cli`, которая автоматически подбирает и устанавливает нужный драйвер.

## Что такое kroko-cli?

`kroko-cli` — это консольная утилита, которая:
- Обнаруживает вашу видеокарту NVIDIA.
- Определяет наиболее подходящий драйвер.
- Автоматически устанавливает и настраивает его.
- Решает проблемы совместимости с ядром системы.

Утилита включена в ROSA Linux начиная с версии R12.5.

## Как использовать

Все команды требуют прав администратора, используйте `sudo`.

### 1. Поиск видеокарты

Сначала определите, какая видеокарта установлена:

```bash
sudo kroko-cli get-gpus
```
Эта команда покажет обнаруженные видеокарты NVIDIA и текущие драйверы.

### 2. Подбор лучшего драйвера

Утилита может сама определить лучший драйвер для вашей системы:

```bash
sudo kroko-cli best-driver
```

### 3. Автоматическая установка

Это основная команда, которая выполнит всю работу:

```bash
sudo kroko-cli autoinstall
```
Она найдёт, скачает и установит нужный драйвер. После завершения установки перезагрузите компьютер.

<div class="info">
  <div class="title">Примечание</div>
  Если утилита не найдёт подходящий драйвер, она не будет вносить никаких изменений в систему.
</div>

## Когда это полезно?

- **После установки ROSA:** если у вас низкое разрешение экрана или нет 3D-ускорения.
- **После обновления системы:** если старый драйвер перестал работать с новым ядром.
- **На ноутбуках с гибридной графикой:** для активации дискретной видеокарты NVIDIA.

## Справка

Для получения полного списка команд выполните:

```bash
kroko-cli help
```
