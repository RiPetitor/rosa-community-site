В этом разделе собраны руководства по работе с проприетарными драйверами NVIDIA в ROSA Linux.

Проприетарные драйверы необходимы для полноценной работы видеокарты: они обеспечивают 3D-ускорение для игр и приложений, аппаратное декодирование видео и доступ к технологиям вроде CUDA.

Здесь вы найдёте информацию по трём основным темам:

- **[Ручная установка](@/docs/02-daily-use/nvidia/01-manual-install.md):** Для полного контроля над процессом.
- **[Автоматическая установка](@/docs/02-daily-use/nvidia/02-kroko-cli.md):** Рекомендованный и самый простой способ с помощью утилиты `kroko-cli`.
- **[Гибридная графика](@/docs/02-daily-use/nvidia/03-hybrid-graphics.md):** Рекомендации для систем с гибридной графикой (Optimus)
- **[Диагностика и устранение неисправностей](@/docs/02-daily-use/nvidia/04-troubleshooting.md):** Что делать, если что-то пошло не так.

Выберите подходящий вам способ. Для большинства пользователей мы рекомендуем автоматическую установку.
