Регулярное обновление системы — залог безопасности и стабильной работы.

## Графический способ

### Центр обновлений

В ROSA Linux есть графический инструмент для обновлений:

1. Откройте меню приложений.
2. Найдите «Индикатор обновлений».
3. Он автоматически проверит наличие всех обновлений в вашей системе.
4. Установите доступные обновления.

## Через терминал

### Проверка обновлений и установка

```bash
sudo dnf update
```

Или сокращённый вариант:

```bash
sudo dnf up
```

### Обновление с автоподтверждением

```bash
sudo dnf up -y
```

<div class="warning">
  <div class="title">Осторожно</div>
  Флаг <code>-y</code> автоматически подтверждает все действия. Используйте его, только если уверены в обновлениях.
</div>

## Что обновляется

- **Пакеты системы** — ядро, библиотеки, утилиты.
- **Приложения** — браузеры, офисные программы, мультимедиа.
- **Драйверы** — при наличии обновлений в репозиториях.

## Обновление ядра

Ядро Linux 6.12 LTS обновляется вместе с системой. После обновления ядра рекомендуется перезагрузить компьютер:

```bash
systemctl reboot
```

<div class="tip">
  <div class="title">Совет</div>
  ROSA сохраняет несколько версий ядра. Если новое ядро работает нестабильно, вы можете выбрать предыдущее в меню загрузчика (GRUB).
</div>

## Автоматические обновления

Для серверов и рабочих станций можно настроить автоматическое обновление:

```bash
sudo dnf install dnf-automatic
sudo systemctl enable --now dnf-automatic.timer
```

Настройки находятся в файле `/etc/dnf/automatic.conf`.

## Очистка после обновлений

Удаление старых пакетов и кэша:

```bash
sudo dnf autoremove
sudo dnf clean all
```

## Возможные проблемы

### Конфликт пакетов

Если обновление прерывается из-за конфликтов:

```bash
sudo dnf distro-sync
```

### Нехватка места

Освободите место и повторите:

```bash
sudo dnf clean packages
```

## Особенности обновления в ROSA Linux

- **Оригинальная пакетная база**: ROSA использует независимую пакетную базу с 2012 года.
- **Поддержка новой аппаратуры**: Благодаря ядру 6.12 LTS и glibc 2.40, система поддерживает современное оборудование.
- **Современные технологии**: systemd 256 обеспечивает надежное управление службами.
