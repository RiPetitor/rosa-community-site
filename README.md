# ROSA Community Site

Неофициальный сайт сообщества ROSA Linux в стиле советской газеты.

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/RiPetitor/rosa-community-site.git
cd rosa-community-site

# Запустить локально (требуется Zola)
zola serve
```

Сайт будет доступен по адресу `http://127.0.0.1:1111`

## Структура проекта

```
content/
├── blog/       # Статьи блога
└── docs/       # Документация
sass/           # Стили (SCSS)
static/         # Статические файлы
templates/      # Шаблоны Zola
```
