# ROSA Community Site

Неофициальная газета сообщества ROSA Linux в духе советской печати.

## Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/RiPetitor/rosa-community-site.git
cd rosa-community-site

# Запустить локально (требуется Zola)
zola serve
```

Газета будет доступна по адресу `http://127.0.0.1:1111`

## Структура редакции

```
content/
├── blog/       # Передовицы и заметки
└── docs/       # Справочные материалы
sass/           # Стили оформления
static/         # Статические ресурсы
templates/      # Шаблоны вёрстки
```

## Как стать автором

Хотите опубликовать статью или дополнить документацию? Читайте [Руководство для авторов](CONTRIBUTING.md).

На каждой странице есть кнопка **«Внести правки»** — она откроет редактор на GitHub.

В блоге и документации есть кнопки **«+ Написать»** и **«+ Добавить страницу»** для создания новых материалов.
