# ROSA Community — Памятка по контенту

Короткое руководство для авторов блога и документации.

## Блог
Файлы: `content/blog/*.md`

Минимальный шаблон:
```
+++
title = "Заголовок"
date = 2026-01-30
description = "Короткое описание для списка"

[taxonomies]
categories = ["Новости"]
tags = ["rosa", "релиз"]

[extra]
author = "Имя автора"
+++

Текст статьи...

<!-- more -->

## Подзаголовок
Основной текст...
```

Скрытые фичи:
- `draft = true` — черновик.
- `slug = "moy-post"` — свой URL.
- `aliases = ["/старый-адрес/"]` — редиректы.
- `extra.featured = true` — пометка «Передовица».

## Документация
Файлы: `content/docs/**/page.md`

Шаблон страницы:
```
+++
title = "Название"
description = "Короткое описание"
weight = 2
+++

Текст...
```

Раздел (каталог):
```
+++
title = "Новый раздел"
description = "Что внутри"
template = "docs-section.html"
weight = 5
+++
```

Скрытые фичи:
- `draft = true` — черновик.
- `slug = "custom"` — свой URL.
- `template = "docs-page.html"` — другой шаблон.

## Оформление
### Callouts
```
<div class="tip">
  <div class="title">Совет</div>
  Текст подсказки.
</div>
```
Доступные блоки: `tip`, `info`, `warning`, `danger`.

### Код
````
```bash
sudo dnf update
```
````

### Таблицы
```
| Команда | Назначение |
| --- | --- |
| dnf update | обновление |
| dnf install | установка |
```

### Изображения
Клади файлы в `static/images/`:
```
![Подпись](/images/filename.png)
```

## Пример (наглядно)
### Блог
Создай `content/blog/primer-gazeta.md`:
```
+++
title = "Первые шаги в ROSA"
date = 2026-02-01
description = "Короткая заметка о том, с чего начать."

[taxonomies]
categories = ["Новости"]
tags = ["rosa", "гайд"]

[extra]
author = "Редакция"
+++

Добро пожаловать!

<!-- more -->

## Установка
Коротко о шагах...
```

Результат: карточка появится в `/blog/`, внутри статьи появится оглавление справа.

### Документация
Создай `content/docs/guides/primer-docs.md`:
```
+++
title = "Памятка по обновлению"
description = "Короткий чек-лист после установки"
weight = 6
+++

## Обновление системы
Откройте терминал и выполните:
    sudo dnf update
```

Результат: страница появится в разделе «Руководства».
